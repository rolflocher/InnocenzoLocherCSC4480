Innocenzo Locher CSC4480 final project README

OVERVIEW: This program allows for the navigation, reading and writing of statistics from the 2018-2019 NBA Season. Bringing together Firestore noSQL database with a user friendly MacOS frontend, a user can view and alter a variety of player and team stats from the 2018-2019 season.

INSTRUCTIONS: Upon launching the program, the four pop-up up windows for conference, division, team and player are available. From left to right, make the appropriate selection and narrow down to your desired document. All fields are editable.
